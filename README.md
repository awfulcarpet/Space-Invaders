# Space Invaders
emulation of the original space invaders arcade cabinet in C and SDL2
![Space Invaders attract mode](res/attract.png)

## build & run
```sh
make run
```

## Controls
 - **C**: insert coin
#### Player 1
 - **Left Arrow**: Move Left
 - **Right Arrow**: Move Right
 - **Space**: Shoot
 - **Enter**: start
#### Player 2
 - **A**: Move Left
 - **D**: Move Right
 - **F**: Shoot
 - **Backspace**: start
